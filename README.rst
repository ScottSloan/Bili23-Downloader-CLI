Bili23 Downloader CLI 命令行版本

下载 Bilibili 视频/番剧/电影/纪录片 等资源

使用说明
========
详细信息请移步至 `项目主页 <https://github.com/ScottSloan/Bili23-Downloader-CLI>`_ 查看

安装：

.. code-block:: console
    
    pip install bili23


下载视频示例：

.. code-block:: console

    $ bili23 --p "BV1fd4y117xT"
